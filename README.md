# 📊 Startup Investment Portfolio Risk Analysis

## 🔹 Overview
This project analyzes three investment portfolios (A, B, and C) under uncertain market conditions using probability-based risk modelling.

## 🔹 Objective
To identify the optimal investment strategy by evaluating expected returns, risk, and sensitivity to changing market probabilities.

## 🔹 Tools Used
- Microsoft Excel
- Data Analysis
- Probability & Risk Modelling
- Data Visualization

## 🔹 Key Insights
- Portfolio B provides the highest expected return ($139,500)
- Portfolio C offers lowest risk but limited growth
- Portfolio B remains optimal across most realistic scenarios
- Results are highly sensitive to high-growth probability

## 🔹 Results
| Portfolio | Expected Payoff | Risk Level |
|----------|----------------|-----------|
| A | $122,500 | Moderate |
| B | $139,500 | High |
| C | $104,500 | Low |

## 🔹 Conclusion
Portfolio B is the best choice for startups seeking aggressive growth, despite higher risk.
